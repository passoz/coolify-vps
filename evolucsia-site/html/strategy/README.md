# Strategy

Pasta interna para estruturar a oferta comercial, a operação e o modelo de entrega da Evolucsia.

Objetivo:

- amadurecer o posicionamento antes de refletir isso na landing
- organizar a oferta como solucao customizada
- descrever como a operacao pode funcionar por cliente
- registrar decisoes e perguntas em aberto
- manter a comunicacao comercial realista enquanto o foco ainda esta sendo definido

Observacao:

- esta pasta foi marcada com `.htaccess` para bloquear acesso em servidores Apache
- se o site for publicado em outro stack, sera preciso criar a regra equivalente no host
