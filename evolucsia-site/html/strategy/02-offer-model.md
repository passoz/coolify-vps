# Modelo de Oferta

## Premissa

A oferta e de solucao customizada.

Nao e:

- licenca de software self-service
- produto aberto para qualquer pessoa configurar
- assinatura de plataforma padrao

E:

- diagnostico do fluxo atual
- desenho da automacao
- implantacao da estrutura
- acompanhamento e ajustes

## O que esta sendo vendido

Uma solucao que pode incluir:

- atendimento automatizado
- triagem inicial
- coleta de dados
- encaminhamento interno
- follow-up
- confirmacoes
- cobranca leve
- lembretes
- organizacao de demandas

## Formato ideal de venda

Servico produtizado com stack proprietaria por tras.

Ou seja:

- o cliente percebe uma solucao profissional pronta para seu contexto
- a implementacao tem componentes reaproveitaveis internamente
- a entrega nao depende de reinventar tudo do zero

## Vantagem desse modelo

- permite cobrar implantacao
- permite cobrar mensalidade de operacao/suporte
- evita a pressao de parecer software de prateleira
- permite adaptar o fluxo por nicho
- encaixa melhor em vendas consultivas

## Estrutura comercial inicial

Modelo simples:

- taxa de implantacao
- mensalidade de manutencao e operacao
- servicos extras quando houver novas automacoes ou integracoes

## Resultado desejado

O cliente deve sentir que contratou:

- uma estrutura que organiza a rotina
- um sistema de atendimento e automacao sob medida
- uma camada inteligente que reduz dependencia operacional
